# AI Spend Auditor

AI Spend Auditor is a web application that helps startups and developers analyze their AI tooling expenses and discover opportunities to reduce unnecessary spending.

The platform evaluates current AI subscriptions, identifies inefficient pricing choices, and provides actionable recommendations with estimated monthly and annual savings.

## Features

- AI tool spending audit
- Savings estimation dashboard
- Personalized optimization recommendations
- Modern responsive UI
- Local form persistence using localStorage
- Lead capture flow
- Multi-tool support:
  - ChatGPT
  - Claude
  - Cursor
  - GitHub Copilot
  - Gemini
  - OpenAI API
  - Anthropic API
  - Windsurf
  - v0

## Tech Stack
